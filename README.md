# Elm

1. Init Elm project:
    `elm init`

2. Elm Repl:
    `elm repl`

3. `.elm` to `.js`:
    `elm make src/HomePage.elm --output elm.js`

4. Install dependency:
    `elm install elm/http`

5. Build with server -> http://localhost:8000:
    `elm reactor`

6. Test:
    `npm install elm-test -g`
    `elm-test init`

7. Run Test: `elm-test`
